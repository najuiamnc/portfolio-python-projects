# 🎮 Portfolio Python Projects

> Mini-jogos desenvolvidos em Python para praticar lógica de programação, estruturas de controle e funções.

---

## 📁 Projetos

### 🎯 Quiz Game
Jogo de perguntas e respostas no terminal. O usuário responde perguntas e o programa contabiliza o score ao final.
- Listas, funções e lógica de repetição

### 🔢 Number Guesser
O programa sorteia um número e o jogador tenta adivinhar. A cada tentativa recebe uma dica se o número é maior ou menor.
- Condicionais, loops e entrada do usuário

### ✂️ Rock Paper Scissors
Pedra, papel e tesoura contra o computador. O computador escolhe aleatoriamente e o resultado é calculado automaticamente.
- `random`, condicionais e organização de funções

### 🗺️ Choose Your Own Adventure
Jogo de aventura baseado em escolhas do jogador. A narrativa muda conforme as decisões tomadas.
- Estruturas de decisão, criatividade e storytelling em código

## ▶️ Como executar

**Pré-requisito:** Python 3 instalado

```bash
# Clone o repositório
git clone https://github.com/najuiamnc/portfolio-python-projects

# Entre na pasta
cd portfolio-python-projects

# Execute o projeto desejado
python quiz_game.py
python number_guesser.py
python rock_paper_scissors.py
python choose_your_own_adventure.py
```
